城市绿化信息管理系统（Web全栈项目）
技术栈：MySQL 8.0 + Node.js + React + Tomcat + ECharts

面向政府环保部门，设计并开发城市绿化信息管理系统，实现对街道清扫、洒水作业、落叶打捞、杂草清理等业务的统一管理
基于 MySQL 8.0 完成数据库建模，设计包含员工、环卫站、街道、传感器、作业记录等10+数据表，构建完整业务数据体系
后端基于 Java + Tomcat 部署，提供 RESTful 接口，实现数据的增删改查及业务逻辑处理
前端使用 React 构建管理系统界面，实现表格管理、表单交互、树形结构（水域层级管理）等复杂交互功能
实现环卫站人员调度与任务分配机制，支持多角色协同完成作业任务
构建环境监测模块，结合传感器数据（湿度、温度等）实现数据驱动的洒水作业决策
使用 ECharts 实现多维度数据可视化（饼图、柱状图等），辅助管理决策
通过随机函数与批量插入模拟5000+业务数据，提升系统测试覆盖率与稳定性
